# d3 fun (damentals) 

Chronicle of modest d3 beginnings.

## Features
 - static content example directories compile
 - angular-hipster-seed boilerplate

## Examples
 - css bar chart
 - svg bar chart (with axis)
 - pie chart


## Quickstart

```sh
git clone https://github.com/ebake/d3-fun-damentals.git
npm install
bower install
gulp compile
gulp serve
```
